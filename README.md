#  Digitools Platform

Welcome to **Digitools Platform** — your all-in-one solution for powerful digital tools designed to simplify workflows, boost productivity, and enhance user experience.


##   About the Project

**Digitools Platform** is a modern web-based platform that provides a collection of smart tools to automate tasks, manage workflows, and improve efficiency. Whether you're a developer, designer, or entrepreneur, Digitools helps you get more done with less effort.


⚙️ 🛠️ Technologies Used

*  React.js
*  Tailwind CSS & DaisyUI
*  JavaScript (ES6+)
*  React-Toastify (NPM Package)
*  JSON (for product data)


##  Key Features

### 1.  User-Friendly Interface

Clean, modern, and responsive UI that ensures a smooth experience across all devices.

### 2.  Add to Cart Functionality

Easily add products to your cart with real-time updates and interactive feedback.

### 3.  Remove from Cart

Remove items from the cart instantly with smooth UI updates.

### 4.  Product & Cart Button Functionality

Dynamic buttons that update based on cart state (e.g., add/remove, selected status), improving user interaction and experience.
